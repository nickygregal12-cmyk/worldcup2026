# Agent Control

This folder is the durable first-read layer for future agents working on the Euro 2028 Predictor repo.

Read `00-READ-FIRST.md` before using historical docs or chat memory. The files here consolidate permanent rules, scope guards, workflow discipline, visual review expectations, database safety and document authority.

These files do not delete, archive or replace historical evidence. They tell agents what to read first and how to resolve uncertainty.
