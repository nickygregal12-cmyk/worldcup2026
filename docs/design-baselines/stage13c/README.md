# Stage 13C design baselines

Deterministic reference captures for the two separate knockout experiences.

- `bracket-*`: the permanent winner-only Original Predictor bracket in the predicted context.
- `ko-*`: the real-fixture KO Predictor match centre with 90-minute scores, advancing team, method and KO jokers.

Each experience is captured at 380×844, 768×1024 and 1200×1000 in light and dark themes. These references protect the visual boundary between predicted and real-fixture contexts in later Stage 13 work.
