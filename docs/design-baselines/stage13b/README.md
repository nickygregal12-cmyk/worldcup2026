# Stage 13B Groups predictor baselines

Reference views for the mobile-first Groups predictor at the Design Charter sizes:

- 380 × 844, light and dark;
- 768 × 1024, light and dark;
- 1200 × 1000, light and dark.

The fixture is explicitly provisional and uses realistic sample nations only to review layout, score controls, state chips, local circle flags and joker presentation. It is not tournament data. The implementation remains data-driven through ISO codes and slot references.
