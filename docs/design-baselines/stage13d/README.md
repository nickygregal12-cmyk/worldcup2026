# Stage 13D design baselines

Deterministic signed-in reference captures for the integrated league and result journeys.

- `leagues-*`: private-league landing, separate competition summaries, shared member selector and responsive standings.
- `results-*`: canonical result status, live context, separate leaderboards and separate points journeys.

Each journey is captured at 380×844, 768×1024 and 1200×1000 in light and dark themes. The fixture contains realistic member volume, all 51 canonical match rows, one live knockout match, one manual-review result and one corrected result.

These are functional-integration references rather than the final Euro 2028 artwork pass.
