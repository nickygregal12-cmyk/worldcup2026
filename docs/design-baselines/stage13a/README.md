# Stage 13A design baselines

Reference views are stored at the Design Charter widths in both themes:

- 380 × 844 mobile;
- 768 × 1024 tablet;
- 1200 × 1000 desktop.

They establish the Stage 13A layout, hierarchy, blue direction and five-position mobile navigation. The deployed staging preview remains the approval surface for final colour tuning. Later Stage 13 batches must compare their screens against these references and update them deliberately rather than allowing visual drift.
